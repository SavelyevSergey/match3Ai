## 7. План разработки

### 7.1. Этапы разработки

#### 7.1.1. Pre-production (2-3 недели)
**Цели:**
- Финализация ГДД
- Создание прототипа базовой механики
- Тестирование core gameplay loop

**Задачи:**
- [ ] Реализация Grid System (переменный размер)
- [ ] BFS алгоритм для Match Detection
- [ ] Базовая система ввода (свайп)
- [ ] Простейший уровень (8×8, без препятствий)
- [ ] Placeholder графика

**Deliverables:**
- Играбельный прототип
- Подтверждение "fun factor"

#### 7.1.2. Production - Alpha (6-8 недель)
**Цели:**
- Реализация всех основных механик
- 30-40 уровней
- Базовый UI/UX

**Задачи:**
- [ ] Все типы бустеров + комбинации
- [ ] Все типы препятствий
- [ ] State Machine для игры
- [ ] 4 типа уровней (Collection, Clear, Drop, Score)
- [ ] Система звёзд и очков
- [ ] Level Editor (инструмент для создания уровней)
- [ ] Shuffle система + Hint
- [ ] Интеграция графики (ассеты)
- [ ] Базовый аудио

**Deliverables:**
- Alpha build с 30+ уровнями
- Внутреннее тестирование

#### 7.1.3. Production - Beta (4-6 недель)
**Цели:**
- Полировка
- 60-80 уровней
- Балансировка
- Оптимизация

**Задачи:**
- [ ] Финальная графика и анимации
- [ ] Полный аудио (музыка + SFX)
- [ ] UI/UX полировка
- [ ] 60-80 уровней с кривой сложности
- [ ] Тестирование Win Rate
- [ ] Оптимизация производительности (60 FPS на средних устройствах)
- [ ] Bug fixing

**Deliverables:**
- Beta build для тестирования
- Focus группы / бета-тестеры

#### 7.1.4. Soft Launch (2-4 недели)
**Цели:**
- Тестирование на реальных пользователях
- Сбор метрик
- Быстрые итерации

**Регионы:**
- 1-2 небольших рынка (например, Филиппины, Канада)

**Метрики:**
- Retention (D1, D7, D30)
- Session length
- Level completion rate
- Win rate по уровням
- Churn points (где игроки уходят)

**Задачи:**
- [ ] Интеграция аналитики (Unity Analytics / Firebase)
- [ ] A/B тестирование сложности уровней
- [ ] Корректировка баланса на основе данных
- [ ] Bug fixing

#### 7.1.5. Global Launch (1 неделя подготовки)
**Цели:**
- Выпуск на все рынки
- Маркетинг
- Поддержка

**Задачи:**
- [ ] Финальная оптимизация
- [ ] Подготовка App Store / Google Play страниц
- [ ] Press-kit
- [ ] Community management (соцсети, форумы)

#### 7.1.6. Live Ops (постоянно)
**Цели:**
- Удержание игроков
- Монетизация
- Рост аудитории

**Активности:**
- Новые уровни (20 каждые 2 недели)
- События (временные челленджи)
- Сезонный контент (праздники)
- Баланс-патчи
- Новые фичи (например, PvP, кланы)

---

### 7.2. Roadmap

#### 7.2.1. Основные фичи по этапам

**Pre-production:**
- ✅ Core Match-3 механика
- ✅ Базовый Grid
- ✅ Простейший Level

**Alpha:**
- ✅ Все бустеры
- ✅ Все препятствия
- ✅ 4 типа уровней
- ✅ Система звёзд
- ✅ 30 уровней
- ✅ Базовый UI

**Beta:**
- ✅ Финальная графика
- ✅ Полный аудио
- ✅ 80 уровней
- ✅ Балансировка
- ✅ Оптимизация

**Soft Launch:**
- ✅ Аналитика
- ✅ A/B тестирование
- ✅ Корректировки баланса

**Global Launch:**
- ✅ 100 уровней
- ✅ Полный контент
- ✅ Маркетинг

**Post-Launch (Live Ops):**
- Месяц 1-2: Стабилизация, багфиксы, новые уровни
- Месяц 3-4: События, сезонный контент
- Месяц 5-6: Новая мета-механика (например, строительство города как в Homescapes)
- Месяц 7+: PvP, кланы, социальные фичи

#### 7.2.2. Контент-план

**Регулярный контент:**
- **Уровни:** 20 новых каждые 2 недели
- **События:** 1 событие в месяц (длительность 7-14 дней)
- **Сезоны:** смена темы каждые 3 месяца

**Типы событий:**
- Временные уровни с уникальными наградами
- Турниры (кто наберёт больше очков)
- Дневные челленджи
- Праздничные события (Хэллоуин, Новый год и т.д.)
