## 7. План разработки

### 7.1. Этапы разработки

#### 7.1.1. Pre-production
**Цели:**
- Финализация ГДД
- Создание прототипа базовой механики
- Тестирование core gameplay loop

**Задачи:**
- [ ] Реализация Grid System (переменный размер)
- [ ] BFS алгоритм для Match Detection
- [ ] Базовая система ввода (свайп)
- [ ] Простейший уровень (8×8, без препятствий)
- [ ] Placeholder графика

**Deliverables:**
- Играбельный прототип
- Подтверждение "fun factor"

#### 7.1.X. Production - Vertical Slice
**Цели:**
- Создание 1-2 референсных уровней
- Финализация визуального стиля
- Финализация аудио
- Определение стандартов качества

**Задачи:**
- [ ] 1-2 полностью полированных уровня
- [ ] Финальная графика и анимации
- [ ] Полный аудио для уровней
- [ ] UI/UX для игрового процесса
- [ ] Документация стандартов уровней

**Deliverables:**
- Vertical Slice build
- Согласованные стандарты качества

#### 7.1.2. Production - Alpha
**Цели:**
- Реализация всех основных механик
- 35 уровней
- Базовый UI/UX

**Задачи:**
- [ ] Все типы бустеров + комбинации
- [ ] Все типы препятствий
- [ ] State Machine для игры
- [ ] 4 типа уровней (Collection, Clear, Drop, Score)
- [ ] Система звёзд и очков
- [ ] Shuffle система + Hint
- [ ] Интеграция графики (ассеты)
- [ ] Базовый аудио

**Deliverables:**
- Alpha build с 35 уровнями
- Внутреннее тестирование

#### 7.1.3. Production - Beta
**Цели:**
- Полировка
- 55-60 уровней
- Балансировка
- Оптимизация

**Задачи:**
- [ ] Финальная графика и анимации
- [ ] Полный аудио (музыка + SFX)
- [ ] UI/UX полировка
- [ ] Создание уровней вручную через Unity Inspector
- [ ] 55-60 уровней с кривой сложности
- [ ] Тестирование Win Rate
- [ ] Оптимизация производительности (60 FPS на средних устройствах)
- [ ] Bug fixing

**Deliverables:**
- Beta build для тестирования
- Focus группы / бета-тестеры

#### 7.1.4. Soft Launch
**Цели:**
- Тестирование на реальных пользователях
- Внутреннее тестирование стабильности
- Bug fixing на основе обратной связи

**Регионы:**
- 1-2 небольших рынка (например, Филиппины, Канада)

**Задачи:**
- [ ] Bug fixing
- [ ] Финальная полировка

#### 7.1.5. Global Launch
**Цели:**
- Выпуск на все рынки
- Маркетинг
- Поддержка

**Задачи:**
- [ ] Финальная оптимизация
- [ ] Подготовка App Store / Google Play страниц
- [ ] Press-kit
- [ ] Community management (соцсети, форумы)

**Post-Launch:**
**Примечание:** Live Ops в классическом смысле отменён из-за архитектурных ограничений (ScriptableObject без Asset Bundles). Все 80 уровней доступны с релиза.

**Цели:**
- Стабильность и качество
- Удержание игроков

**Активности:**
- Bug fixing
- Баланс-патчи
- Оптимизация производительности
- Сезонные визуальные темы (без добавления уровней)
- Расширение контента возможно только с пересборкой билда

---

### 7.2. Roadmap

#### 7.2.1. Основные фичи по этапам

**Pre-production:**
- ✅ Core Match-3 механика
- ✅ Базовый Grid
- ✅ Простейший Level

**Alpha:**
- ✅ Все бустеры
- ✅ Все препятствия
- ✅ 4 типа уровней
- ✅ Система звёзд
- ✅ 30 уровней
- ✅ Базовый UI

**Beta:**
- ✅ Финальная графика
- ✅ Полный аудио
- ✅ 80 уровней
- ✅ Балансировка
- ✅ Оптимизация

**Soft Launch:**
- ✅ 70 уровней
- ✅ Стабилизация
- ✅ Bug fixing

**Global Launch:**
- ✅ 80 уровней (4 эпизода)
- ✅ Полный контент
- ✅ Маркетинг

- Месяц 1-2: Стабилизация, багфиксы
- Месяц 3-4: Баланс-патчи, оптимизация
- Месяц 5-6: Сезонные визуальные темы
- Месяц 7+: Расширение контента (только с пересборкой билда)

#### 7.2.2. Контент-план

**Регулярный контент:**
- **Уровни:** Новые уровни на регулярной основе
- **События:** Периодические события с уникальными наградами
- **Сезоны:** Смена темы по мере необходимости

**Типы событий:**
- Временные уровни с уникальными наградами
- Турниры (кто наберёт больше очков)
- Дневные челленджи
- Праздничные события (Хэллоуин, Новый год и т.д.)
