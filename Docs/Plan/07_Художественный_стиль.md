# 7. Художественный стиль

[◀ Назад к оглавлению](README.md) | [▶ Следующий раздел: Аудио](08_Аудио.md)

---

#### 5.1.1. Art Style
- **Стиль:** Casual Cartoon (яркий, дружелюбный)
- **Целевая аудитория:** широкая (семейная игра)
- **Референсы:**
  - Candy Crush: яркие элементы с градиентами
  - Homescapes: мягкий cartoon с деталями
  - Toon Blast: упрощённый, читаемый стиль

#### 5.1.2. Цветовая палитра
**Игровые элементы (основные):**
- Красный: #FF3B3B
- Синий: #3B88FF
- Зелёный: #3BFF6E
- Жёлтый: #FFD93B
- Фиолетовый: #B03BFF
- Оранжевый: #FF8C3B

**UI:**
- Фон: светлые пастельные тона (#F5F5DC, #E6F3FF)
- Акценты: яркие цвета для кнопок
- Текст: тёмно-серый (#333333) для читаемости

#### 5.1.3. Визуальные референсы
- **Ассеты:** [OpenGameArt.org - Match-3 Assets](https://opengameart.org/art-search-advanced?keys=match+3)
- **Цветовые схемы:** [Coolors.co - Palette Generator](https://coolors.co/)

---

### 5.2. Ассеты

#### 5.2.1. Игровые элементы
**Типы элементов:**
- 6-7 базовых типов (драгоценные камни или конфеты)
- Варианты бустеров (полосатый, обёрнутый, бомба)
- Размер: 128×128px (рекомендуется для Unity)

**Источники бесплатных ассетов:**
- [Kenney.nl - Game Assets](https://kenney.nl/assets/puzzle-pack-2) - CC0 License
- [Freepik - Game Elements](https://www.freepik.com/free-photos-vectors/match-3) - Free License (с атрибуцией)
- [Itch.io - Free Assets](https://itch.io/game-assets/free/tag-match3) - различные лицензии

#### 5.2.2. Препятствия
- Лёд: полупрозрачный синий с текстурой
- Камень: серый, твёрдый
- Коробка: деревянная текстура
- Желе: полупрозрачный с эффектом "желе"
- Цепь: металлические звенья

**Источники:**
- [OpenGameArt - Obstacles](https://opengameart.org/content/match-3-obstacles)

#### 5.2.3. Персонажи
- **Главный герой/помощник:** (опционально) cartoon персонаж для сюжета
- **Стиль:** дружелюбный, простой дизайн
- **Источники:**
  - [Craftpix - Free Characters](https://craftpix.net/freebies/free-game-character-sprites/)

#### 5.2.4. Окружение
- Фоны для карты уровней (тематические зоны: лес, горы, пляж и т.д.)
- Фоны для игрового поля (нейтральные, не отвлекающие)

**Источники:**
- [GameArt2D - Free Backgrounds](https://www.gameart2d.com/freebies.html)

#### 5.2.5. Анимации

**Ключевые анимации:**
- Свап элементов (0.3s)
- Уничтожение элемента (particle burst, 0.4s)
- Падение элемента (smooth fall, 0.5s)
- Появление элемента (scale from 0, 0.3s)
- Активация бустера (flash + expand, 0.5s)

**VFX эффекты:**
- Взрыв бустера: частицы, яркая вспышка
- Комбо-индикатор: звёздочки, искры
- Match подсветка: glow эффект

**Источники:**
- [Unity Particle Pack](https://assetstore.unity.com/packages/essentials/tutorial-projects/unity-particle-pack-127325) - Free
- [OpenGameArt - VFX](https://opengameart.org/art-search-advanced?keys=particle+effect)

## 6. Аудио

### 6.1. Музыка

#### 6.1.1. Фоновая музыка
**Зоны/эпизоды:**
- Эпизод 1-20: лёгкая, весёлая мелодия (upbeat)
- Эпизод 21-40: чуть более динамичная
- Эпизод 41+: эпическая, приключенческая

**Требования:**
- Loop-friendly (бесшовное зацикливание)
- Tempo: 120-140 BPM
- Не отвлекает от геймплея

**Источники бесплатной музыки:**
- [FreeMusicArchive - Game Music](https://freemusicarchive.org/genre/Video_Game) - CC License
- [Incompetech - Royalty Free](https://incompetech.com/music/royalty-free/music.html) - CC BY 4.0
- [OpenGameArt - Music](https://opengameart.org/art-search-advanced?keys=&field_art_type_tid%5B%5D=12) - различные лицензии

#### 6.1.2. Джинглы
**Победа:**
- Длительность: 3-5 секунд
