## 1.1. Краткое описание игры

### Основная информация
- **Жанр:** Match-3 Puzzle
- **Основная механика:** Комбинирование трёх и более одинаковых элементов для очистки игрового поля
- **Платформа:** Android
- **Движок:** Unity 2022.3.62f3
- **Язык программирования:** C#
- **Ограничения:** без сторонних плагинов

### Ключевые особенности
- Классическая механика Match-3 со свайп-управлением
- Система бустеров и их комбинаций
- Разнообразные типы препятствий
- Прогрессивная сложность уровней
- Система достижений и звёздных рейтингов

---

## 1.2. Референсы

### Анализ конкурентов

#### Candy Crush Saga
**Что берём:**
- Базовая механика Match-3 со свайпами
- Система бустеров (полосатые, обёрнутые, цветные бомбы)
- Структура эпизодов по 15-20 уровней
- Визуально яркий casual стиль

**Что улучшаем:**
- Более быстрый геймплей (меньше ожиданий анимаций)
- Честная сложность без агрессивной монетизации
- Больше разнообразия в типах уровней

#### Homescapes / Gardenscapes
**Что берём:**
- Комбинированные цели на уровнях
- Разнообразие препятствий
- Мягкий cartoon стиль
- Прогрессивное введение механик

**Что улучшаем:**
- Фокус на core gameplay без мета-игры (на старте)
- Более чистый и читаемый визуальный стиль
- Оптимизированная производительность для слабых устройств

#### Toon Blast
**Что берём:**
- Упрощённая визуальная стилистика
- Быстрый темп игры
- Чёткие и понятные цели

**Что улучшаем:**
- Больше глубины в механиках (комбинации бустеров)
- Более проработанная кривая сложности

## 1.3. Ключевые отличия в механиках

### Уникальные особенности (если планируются)

1. **Особые комбинации**
   - Расширенная таблица взаимодействий бустеров
   - Уникальные эффекты при комбинировании

2. **Динамическая сложность**
   - Адаптивная генерация элементов на основе прогресса игрока
   - Weighted random для целевых элементов

3. **Технические преимущества**
   - Оптимизация для Android 6.0+
   - 60 FPS на средних устройствах
   - Малый размер билда (< 100 MB)
   - Работа без сторонних плагинов

## Целевая аудитория

- **Возраст:** 12+
- **Платформы:** Android (основная), возможность портирования на iOS
- **Стиль игры:** Casual, для коротких игровых сессий (5-15 минут)
- **Опыт:** как для новичков, так и для опытных игроков в Match-3

---

## Основные цели проекта

1. **MVP (Pre-production):** Подтверждение fun factor базовой механики
2. **Alpha:** Полный набор механик и 35 уровней
3. **Beta:** Балансировка, 55-60 уровней, полировка
4. **Soft Launch:** 70 уровней, стабилизация, финальное тестирование
5. **Global Launch:** Релиз с **80 уровнями** (4 эпизода)
6. **Post-Launch:** Bug fixes, баланс-патчи, оптимизация

> **Примечание:** Live Ops в классическом смысле отменён из-за архитектурных ограничений (ScriptableObject без Asset Bundles). Все 80 уровней доступны с релиза.
